# GraphQL with Nest (Code first approach)

## For node team huddle 09-2021

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Seeds and schema provided

```bash
$ npm run schema:sync
$ npm run seed:run
```
